# BoomingScanner

스피커 부밍을 자동으로 감지하고 안내해주는 소리 보정 도우미입니다.

스피커를 처음 설치하거나 새로운 공간에서 사용할 때, 특정 주파수 대역이 과도하게 울리는 부밍 현상은 많은 사용자에게 어려움으로 다가옵니다. 일반적으로 부밍을 파악하기 위해서는 특정 톤을 재생하고, 마이크로 직접 측정한 뒤, 주파수 그래프를 해석해야 하는데, 이 과정은 초보자에게는 어렵고 숙련자에게도 반복 작업이 될 수 있습니다.
BoomingScanner는 이러한 과정을 간단하게 자동화하여, 사용자가 보다 편하게 음향 환경을 점검하고 보정할 수 있도록 돕기 위해 만들어졌습니다.

## 프로젝트 의의

BoomingScanner는 단순한 음향 보정 도구를 넘어, 스피커와 공간, 그리고 사용자 사이에서 발생하는 복잡한 음향 상호작용을 보다 이해하기 쉽게 풀어내는 것을 목표로 하고 있습니다. 소리를 측정하고, 그 결과를 다시 더 나은 소리로 환원하는 경험을 제공하고자 합니다.

## 기능 소개

### 1. 테스트 신호 재생
    
측정을 위해 필요한 주파수 스윕 또는 특정 대역(예: 60Hz, 120Hz)을 프로그램에서 직접 재생합니다.

### 2. 마이크 입력 기반 실측

PC 내장 마이크 또는 외부 마이크를 사용해 반응을 실시간으로 수집합니다. 이를 통해 공간 음향과 스피커의 특성을 기반 데이터로 확보합니다.

### 3. 부밍 의심 대역 자동 분석

FFT(Fast Fourier Transform)를 이용해 특정 주파수에서 음압이 비정상적으로 상승한 구간을 자동으로 감지합니다. 부밍 가능성이 있는 대역을 수치와 그래프로 확인할 수 있습니다.

### 4. EQ 보정 가이드 제공

문제가 되는 주파수에 대해 “해당 대역을 일정 수준 감쇄해 보세요”, “Q 값을 조정해보세요”와 같은 실용적인 보정 가이드를 제공합니다. 초보자도 쉽게 따라 할 수 있도록 설명하는 것을 목표로 하고 있습니다.

## 목표 사용자

- 스피커 튜닝에 익숙하지 않은 초보자
- 별도의 룸 튜닝 없이 음향을 개선하고 싶은 사용자
- EQ 보정을 직접 하고자 하지만 시작점이 고민되는 사용자
- 반복적인 측정·보정 과정을 자동화하고 싶은 오디오 사용자
